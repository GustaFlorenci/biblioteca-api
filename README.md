# 📚 Biblioteca API

API REST desenvolvida em **Java 17** com **Spring Boot**, criada para praticar arquitetura em camadas, JPA, PostgreSQL e regras de negócio em um sistema de biblioteca.

## 🚀 Tecnologias

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Maven
- Hibernate
